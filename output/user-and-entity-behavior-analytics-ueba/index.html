User And Entity Behavior Analytics (UEBA)

A security process that uses advanced analytics to detect abnormal behavior of users and entities.

Description

User and Entity Behavior Analytics (UEBA) is a cybersecurity technology that focuses on the analysis of user and entity behavior to detect potential security threats. It leverages machine learning, algorithms, and statistical analysis to establish a baseline of normal activities for users and entities, such as devices and applications. Once the baseline is established, UEBA continuously monitors for deviations from this norm. Such deviations can indicate malicious actions, such as insider threats, compromised accounts, or other anomalous behaviors that could lead to data breaches or security incidents. UEBA solutions typically aggregate data from various sources, including log files, network traffic, and user activity, to provide a comprehensive view of behavior patterns. This proactive approach allows organizations to identify and respond to potential threats in real-time, enhancing overall security posture and reducing the risk of breaches.

Examples

  • Detecting a user accessing sensitive data at unusual hours.
  • Identifying a device that is connecting to the network from an unusual geographic location.

Additional Information

  • UEBA can complement traditional security measures like SIEM (Security Information and Event Management).
  • It is particularly effective in identifying insider threats and advanced persistent threats (APTs).
